# VideoForge!

VideoForge is a microservice-based video streaming platform.  
This repository contains the backend services and the frontend UI.

## Project Structure

- `api-gateway` – Spring Boot API Gateway
- `user-service` – Spring Boot service for user management
- `video-service` – Spring Boot service for video management
- `streaming-service` – Spring Boot service for streaming videos
- `ui-service` – Angular frontend
